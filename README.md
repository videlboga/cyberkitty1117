# Smart Summary Telegram Bot

Легковесный Telegram-бот, предназначенный для ежедневной суммаризации сообщений в группах с использованием нейросетей (LLM, например DeepSeek через OpenRouter).

## Архитектура
Бот переписан на чистую и легковесную 3-tier Role System:
- **Superadmin**: Глобальный доступ, возможность принудительного сбора сводок из всех чатов.
- **Admin**: Владелец/администратор группы, привязавший группу к себе. Может запрашивать сводки вручную.
- **User**: Обычные участники чатов, чьи сообщения собираются и обрабатываются.

## Развёртывание
Проект спроектирован для работы из любого сетевого пространства (например `vpnspace`), что необходимо, когда прямой доступ к `api.telegram.org` заблокирован (ТОРГОВЫЙ СБОР / FIREWALL на сервере).

### Требования
- Python 3.9+
- Aiogram < 3.x
- aiohttp, httpx

### Запуск
1. Настройте `config.cfg` и `.env` ключами Telegram и OpenRouter.
2. Пропишите ваш `ADMIN_ID` (ID суперадмина) в `config.cfg`
3. Выполните:
\`\`\`bash
# Пересоздаем окружение
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# Запуск в изолированном пространстве vpnspace
./restart.sh
\`\`\`

## Известные директории и файлы:
- \`main.py\`: Точка входа, обработчик ролей и планировщик.
- \`modules/summary.py\`: Логика форматирования данных и работы с LLM.
- \`modules/db.py\`: Хранилище данных (память + сохранение в JSON-образную структуру).
- \`modules/llm_client.py\`: Интерфейс взаимодействия с API LLM моделей (OpenRouter).
